# Routing Comparison

| Route | Best for | Tradeoff |
| --- | --- | --- |
| Real Claude | Highest compatibility with Claude Code expectations | Requires Anthropic access and billing |
| OpenRouter | Multi-provider routing and model choice | Provider behavior and model IDs may change |
| DeepSeek | Cost-sensitive coding experiments | Compatibility and tool behavior may differ |
| Ollama + LiteLLM | Local/private experiments | Model quality and speed depend on local hardware |

## Required Variables

- `ANTHROPIC_BASE_URL`
- `ANTHROPIC_API_KEY`
- `CLAUDE_MODEL`
